# GADS22 AAD WORKSHOP
Rick and Morty app for the GADS 2022 Associate Android Developer workshop.

Makes use of:

- [x] Hilt for DI
- [x] Paging 3 library
- [x] Remote Mediator
- [x] Room DB
- [x] Retrofit
- [x] Flows
- [x] Coil Image Loader
- [x] Timber
